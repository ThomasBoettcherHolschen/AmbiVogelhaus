# AmbiVogelhaus
Just a little Arduino-Programm to build the LED-Interior of a Birdshouse used as Bedlight for my Daugther.

- Input using 3x4 Matrix https://www.adafruit.com/product/419
- Output WS2801 LED-Strip.
